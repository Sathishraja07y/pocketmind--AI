# 🧠 PocketMind AI – Your Personal Digital Twin

PocketMind AI is an intelligent Android application that learns user behavior and adapts smartphone experiences automatically.

## 🚀 Features

* App Usage Tracking
* Context Detection (Study, Work, Home, Gym, Sleep, Commute)
* Battery Usage Prediction
* Smart Automation Engine
* On-device AI using TensorFlow Lite
* Privacy-focused local data processing
* Real-time Dashboard Analytics

## 🛠 Tech Stack

* Kotlin
* Android Studio
* Jetpack Compose
* Room Database
* TensorFlow Lite
* Python
* TensorFlow/Keras
* Git & GitHub

## 📱 How It Works

1. Collects app usage, battery, sensor, and location data.
2. Builds a digital behavior profile.
3. Predicts user context using AI.
4. Executes smart actions automatically.
5. Displays insights through a modern dashboard.

## 🎯 Project Goal

Create a Personal Digital Twin that understands user habits and proactively improves productivity, battery life, and overall smartphone experience.
